# 영세율첨부서류제출명세서 자동화

**(주)비나우** 부가세 신고용 영세율첨부서류제출명세서 작성 자동화 프로그램

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://your-app-url.streamlit.app)

---

## 📋 개요

매입매출장과 홈택스 자료를 이용해 **영세율첨부서류제출명세서**를 자동 작성합니다.

| 입력 | 출력 |
|---|---|
| 매입매출장 (엑셀) | 영세율첨부서류제출명세서 (엑셀) |
| 환급실적명세서 PDF | 검증_요약 시트 포함 |
| 세금계산서현황 CSV | 외화금액·환율 자동 입력 |

---

## 🗂 파일 구조

```
streamlit_app.py   ← 메인 앱 (Streamlit)

config.json        ← 설정 저장 (자동 생성, Git 제외 권장)

packages.txt       ← 시스템 패키지 (Streamlit Cloud용)
.streamlit/
  config.toml      ← 테마 설정
```

## 🚀 실행 방법

### 로컬 실행
```bash
pip install -r requirements.txt
streamlit run streamlit_app.py
```

### Streamlit Cloud 배포
1. GitHub에 이 저장소를 Push
2. [streamlit.io/cloud](https://streamlit.io/cloud) 접속
3. New app → 저장소/브랜치/파일(`streamlit_app.py`) 선택
4. Deploy

## 📌 작업 순서

### 1️⃣ 기본 설정
기수명, 거래기간, 사업자 정보를 입력하고 저장합니다.  
**매 기수 시작 시 가장 먼저 실행합니다.**

### 2️⃣ 거래처 매핑
거래처별 서류명·비고·통화를 설정합니다.

- 새 거래처가 생기면 행을 추가하고 저장합니다.
- 목록에 없는 서류명·통화는 **서류명·통화 목록 관리** 탭에서 추가하거나, 테이블에 직접 입력합니다.
- 저장하면 페이지를 닫고 다시 열어도 유지됩니다.

### 3️⃣ 1단계: 서식 생성

| 파일 | 필수 | 설명 |
|---|---|---|
| 매입매출장 | ✅ | 영세매출·기타영세만 필터링된 엑셀 |
| 즉시환급 실적명세서 PDF | 선택 | 매장별 여러 파일 한번에 업로드 가능 |
| 사후환급 실적명세서 PDF | 선택 | 매장별 여러 파일 한번에 업로드 가능 |
| 수기전표 PDF | 선택 | 2단계 검증에서 사용 |

업로드한 파일들은 2단계·3단계에서 자동으로 이어받습니다.

### 4️⃣ 2단계: 환급 검증

**Step 1 — 수기전표 vs 반출승인번호 공란 대조**
- 수기전표 PDF는 이미지 스캔본이라 자동 파싱이 안 됩니다.
- 사업장별로 건수·환급액을 **직접 입력**합니다.
- 환급실적명세서의 반출승인번호 공란 건은 자동 파싱됩니다.

**Step 2 — 매입매출장 vs 환급실적명세서 합계 대조**
- 자동으로 처리됩니다.

### 5️⃣ 3단계: 외화금액
세금계산서현황 CSV를 업로드하면 외화금액·환율이 자동 입력됩니다.  
빨간 배경 셀은 매핑 실패 → 직접 입력 필요합니다.

## ⚠️ 주의사항

- **신규 거래처**: 거래처 매핑에 없으면 원화 차이가 납니다.  
  매핑 추가 후 1단계를 재실행하세요.
- **config.json**: 거래처 매핑, 기본 설정 등이 저장됩니다.  
  `.gitignore`에 추가하거나 빈 파일로 커밋하세요 (민감정보 포함 시).
- **파일 용량**: 즉시환급 명세서가 수백 페이지인 경우 처리 시간이 걸릴 수 있습니다.
